import java.util.Scanner;
public class ejercicio4 {
public static void main(String[] args) {

String nombreUsuario, pelicula, comentario;
int opcion;

System.out.print("Por favor indique su nombre: ");
Scanner input = new Scanner(System.in);
nombreUsuario = input.nextLine();

System.out.println("1. Consultar películas disponibles");
System.out.println("2. Alquilar película");
System.out.println("3. Recibir película en la video tienda física");

System.out.println("5. Salir");
System.out.print("Bienvenido, "+nombreUsuario+", por favor elija una opción: ");
opcion = input.nextInt();

switch (opcion) {
case 1:
System.out.println("Busca la película que quieres ver: ");
pelicula = input.nextLine();
System.out.println(pelicula + "se encuentra disponible");
break;
case 2:
System.out.println("Película lista para: " + nombreUsuario);
break;
case 3:
System.out.println("La película se entregará en la sede de la videotienda");
break;
case 4:
System.out.println("Escribe tus comentarios, recuerda ser respetuoso:");
comentario = input.nextLine();
System.out.println("Muchas gracias, lo tendremos en cuenta.");
break;
case 5:
System.out.println("Usted ha finalizado el menú");
break;
default:
System.out.println("Por favor elija una opción válida");
break;
}
}

Para cada cliente, se tienen las opciones de compra de producto, consulta de precios por producto y devoluciones en caso de que se presenten.
Solución:
import java.util.Scanner;

public class ejercicio5 {
public static void main(String[] args) {
String nombre, producto;
int opcion;
Scanner input = new Scanner(System.in);

System.out.println("Bienvenido a la farmacia MiSalud");
System.out.print("Por favor ingrese su nombre: ");
nombre = input.nextLine();
System.out.println("Hola " + nombre + ", elige una opción:");
System.out.println("1. Compra de producto");
System.out.println("2. Consulta de precios");
System.out.println("3. Devoluciones");
System.out.println("4. Salir");
opcion = input.nextInt();
switch (opcion){
case 1:
System.out.print("Ingresa el producto que solicitas: ");
producto = input.nextLine();
System.out.print(producto + " Posee las siguientes indicaciones: No consumir en ayunar, no mezclar con lácteos.");
case 2:
System.out.print("Ingresa el producto a consultar precio: ");
producto = input.nextLine();
System.out.print(producto + " posee un valor de $20.000");
break;
case 3:
System.out.println("En este momento esta opción virtual está en mantenimiento, por favor comunícate" + " a la línea directa de devoluciones 604xxxxxxx");
break;
case 4:
System.out.println("Gracias por visitar droguería MI SALUD, siempre será un gusto");
break;
default:
System.out.println("Ingresa una opción existente");
break;
}
}
}

PUNTO #2

Enunciado:
Se requiere un programa que pueda crear la tabla de multiplicar del número que el usuario indique por medio del ciclo Para; esta debe ser impresa del 1 al 10.
Solución:
import java.util.Scanner;
public class ejercicio4 {
public static void main(String[] args){
Scanner input = new Scanner(System.in);
int numero;
System.out.print("Por favor ingrese un número: ");
numero = input.nextInt();

for(int i = 1; i <= 10; i = i + 1){
System.out.print(numero+" * "+i+" = "+(numero*i)+"\n");
}
}
}

PUNTO #3

Enunciado:
Utilizando el ciclo que usted desee, crear un menú de ejecución infinita, que capture nombre, salude, o salga del sistema, hasta que el usuario desee terminar dicho ciclo.
Solución:
import java.util.Scanner;

public class ejercicio5 {
public static void main(String[] args) {
int opcion;
Scanner sc = new Scanner(System.in);
do {
System.out.println("Menú de usuario");
System.out.println("1. Captura nombre");
System.out.println("2. Saludar persona");
System.out.println("3. Salir del sistema");
System.out.println("");
System.out.print("Ingresa una opción: ");
opcion = sc.nextInt();
}while (opcion != 3);
}
}

public class ejercicio2 {
